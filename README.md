# Student Management System #

## Project Overview
This project demonstrates a Student Management System using SQL. It manages student details, subjects, and marks. The project includes table creation, sample data, joins, aggregate functions, and views.

## Technologies Used
- SQL
- PostgreSQL

## Features
- Student Management
- Subject Management
- Marks Management
- CRUD Operations
- Joins
- Aggregate Functions
- Views
